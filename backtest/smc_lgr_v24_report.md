### 最终参数
| 参数 | 值 | 说明 |

| swing_window | 3 | 激进摆动检测, ~3根K线 |
| adx_threshold | 20 | 趋势确认 |
| fvg_window | 5 | 前后各5根找FVG确认 |

### 核心发现
1. **swing_window=3** 是成功关键 — 更多摆动点→更多掠夺事件
2. **FVG确认** 比OB有效 — FVG密度是OB的30倍
3. **BOC模式不可用** — 结构突破+FVG组合PF<1.0
4. **无需EMA过滤** — LGR是反转模式, EMA限制反而扼杀信号
5. **4h趋势 "any"** — 反转模式不需要趋势同向
